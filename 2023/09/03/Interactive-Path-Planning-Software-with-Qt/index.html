Path planning based on drawing (Qt)

发表于2023-09-03|更新于2023-09-03
|阅读量:

  1. What problem I am trying to solve?
    Try to visualize the path planning processes which are based on BFS,DFS,Dijkstra,Prim algorithm. Meanwhile, corresponding scale graphics can be automatically drawn on the canvas by dealing with satellite images.
    可视化基于BFS,DFS，DIJ,PRIM算法的路径规划。同时根据卫星图像自动生成图像。

  2. Why is this important?
    It could help visualize these abstract algorithms, which could help us understand these algorithms and help us debug. What’s more, it could contribute to education.Able to automatically extract road information using satellite graphics.
    教育和学术研究支持：该项目为教育领域和学术研究提供了重要的支持。通过可视化路径规划算法，它能够帮助学生更好地理解和学习复杂的算法概念。同时，它为研究人员提供了一个工具，可以用于开展算法性能研究和比较不同算法的效率和效果。
    实际应用和规划支持： 该项目不仅仅是一个教育工具，还具有实际应用的潜力。它可以用于城市规划、交通管理和应急响应等领域。通过卫星图像处理，自动提取道路信息，可以在实际环境中帮助决策者更好地理解道路网络，并进行更有效的规划。
    交互式学习：项目的交互性是其重要特征之一。它不仅可以展示路径规划过程，还允许用户干预和探索不同的规划策略。这种交互式学习方法可以激发学生的兴趣，提高他们的参与度，更好地理解算法的原理和应用
    深度学习与图像处理： 项目涉及深度学习和图像处理领域，这两者都是当今科学和工程领域中备受关注的研究方向。通过将这些领域融合到路径规划中，不仅提高了算法的智能性，还促进了交叉学科研究。

  3. How do I understand this problem and how do I formulate it into a math/CS problem?
    这个问题包括很多领域的知识，如图论、图像处理和人工智能，地理信息系统

    为了解决问题，将现实问题细化，然后用计算机处理：

  • 地图表示：我们使用图的数据结构来表示地图，其中节点表示地点，边表示连接地点的道路，同时考虑到某些道路是单向的，因此我们需要同时处理有向图和无向图
  • 地图绘制：我们需要创建一个用户友好的界面，允许用户使用鼠标的点击事件在画布上添加和连接点和线，以及选择它们。
  • 路线选择：我们需要实现多种路径规划算法，包括BFS、Dijkstra、Prim和DFS等，以便根据用户选择的算法在地图上找到最佳路径。
  • 路线可视化绘制：我们需要实现多种路径规划算法，包括BFS、Dijkstra、Prim和DFS等，以便根据用户选择的算法在地图上找到最佳路径。
  • 卫星图像的道路识别：我们计划使用深度学习神经网络进行卫星图像处理，通过训练识别道路区域的像素。一旦识别出道路像素，我们将把它们转化为地理坐标，并在地图上绘制出这些道路。

  1. What’s the current solution? And my solution?
    当前市面上软件面临的问题：

    • 教育与学术支持方面：不能满足用户随意绘制图像、提供高交互性的演示和可视化，以及学习复杂算法概念的需求。
    • 卫星图像方面：虽然存在一些图像识别和分析工具，但它们通常缺乏与路径规划的无缝集成和综合性解决方案

    我的方案：

    • 综合性解决方案： 你的项目提供了一个综合性的解决方案，将路径规划、图像处理和交互式学习融合在一个应用程序中。这种综合性使得用户能够一站式完成路径规划和卫星图像处理，无需切换到不同的工具。
    • 可视化和教育： 你的项目强调了可视化和教育价值。它允许用户实时观察路径规划算法的工作过程，这对于教育和学术研究非常有帮助。同时，用户可以通过交互式学习更好地理解和掌握这些算法。
    • 实际应用： 与传统的路径规划工具不同，你的项目具有实际应用潜力。通过卫星图像处理，它可以自动提取道路信息，为城市规划、交通管理和其他应用提供有用的数据。
    • 深度学习整合： 项目整合了深度学习和图像处理技术，提高了卫星图像处理的智能性。这使得它能够识别道路和提供更准确的地理信息。

  2. What challenges do I met and how do I solve the problems

  • 精确数学计算的重要性： 在图形绘制中，精确的数学计算对于避免几何关系偏差至关重要。通过使用向量知识代替三角函数知识，可以提高绘图的精确性和稳定性。（*）
  • Qt界面与逻辑的对接： 学习如何使用Qt的信号和槽机制是克服界面与逻辑对接问题的关键。这种机制有助于将事件从界面传递到主要逻辑，并实时更新UI，同时保持代码的清晰和模块化。
  • 动态过程的实现与计时器： 动画效果的实现依赖于计时器。理解计时器的工作原理和如何管理时间线可以让你轻松创建基本动画效果。但是却发现决定动画的时间并不随动画暂停而暂停，因此要注意动画停止后的初始化问题（*）
  • 界面和算法的对接挑战： 在团队合作中，代码的规范性和相互之间的沟通至关重要。确保对接的算法具备明确定义的输入和输出形式，有助于简化界面与算法之间的对接过程

具体做了啥？

  • Designed a software where users could draw points and lines on the canvas based on Qt framework.Users could select points and lines and modify the weight of edges and graphics can be transformed between directed and
    undirected graphs.
  • Every operation has a log record and users can also save the already drawn graphics.
  • Implemented the BFS,DFS,Dijkstra,Prim algorithm based on already drawn points and lines.Visualized the traversal process of these algorithms. -Assisted teachers in completing visual teaching of algorithms in the class
  • Based on D-LinkNet, road masks could be formed from satellite images.After applying Gaussian blur to themasks,Road endpoint coordinates on satellite images can be extracted based on HoughLinesP.
  • After merging endpoints and interaction points that are very close to each other, corresponding scale graphics can be automatically drawn on the canvas.Based on these graphs, simple path planning can be carried out.
